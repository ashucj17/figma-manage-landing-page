Manage Landing Page

A fully responsive Manage Landing Page built using modern frontend techniques.
This project focuses on clean UI, accessibility, and responsive design.

Live Demo
https://ashucj17.github.io/figma-manage-landing-page/

🛠️ Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla JS)
- A11Y Slider (Accessible Carousel)

✨ Features
- Fully responsive (Mobile → Desktop)
- Accessible navigation (ARIA attributes used)
- Mobile hamburger menu with toggle
- Smooth icon transition (Hamburger ↔ Close)
- Overlay background for mobile menu
- Testimonial carousel (A11Y Slider)
- Clean and reusable CSS architecture
- Utility-first styling approach

📱 Responsive Design
- Mobile-first approach
- Breakpoints using em units
- Flexible layout using:
- flexbox
- grid
- clamp()
  
🎯 Key Learnings
- Handling z-index & stacking context
- Building accessible UI using:
- aria-expanded
- aria-controls
- Creating reusable CSS variables
- Debugging layout issues in responsive design
- Managing state using vanilla JavaScript
